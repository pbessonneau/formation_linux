\section{Accéder au terminal}

\subsection{Introduction}

  \subsection{Terminal/Console}

	Le Terminal est souvent nécessaire sous \Linux. Plus précisement il est souvent plus facile de passer
	par le Terminal pour faire des réglages qu'en mode graphique.
	
	C'est à la fois un plus et le talon d'Achille de \Linux.
		


	Le Terminal est souvent dans le répertoire des outils systèmes (KDE, Cinnamon,\dots). Il s'appelle \emph{Terminal} dans Gnome3. 
	
	On peut le lancer et ouvrir plusieurs sessions (les logiciels supportent quasiment tous plusieurs
	sessions). Chaque session étant indépendante.
	
	La console est aussi accessible sur les bureaux CTRL+ALT+F1, CTRL+ALT+F2, \dots,  CTRL+ALT+F5.	Initialement l'interface graphique était sur le bureau 6 ou 7 maintenant on peux le trouver sur le bureau 1 ou 2.
	

\section{Commandes primaires}

  \subsection{Raccourcis clavier}

	Vous pouvez utiliser la flêche haute pour rappeler des commandes déjà soumises.
	\vspace{1em}
	Si vous tapez le début d'une commande ou d'un fichier vous pouvez compléter avec Tab.
		

  \subsection{Terminal/Console}

	Les commandes primaires seront~:
	\begin{description}
		\item[cd] change de répertoire. Pour changer de répertoire il suffit de donner son nom, vide vous retournez à votre répertoire personnel.
		\item[ls] liste les fichiers et répertoires. Pour avoir les détails sur les fichiers utiliser \emph{ls -l}
		\item[cp] copie les fichiers et répertoires. 
		\item[mv] copie les fichiers et répertoires. 
		\item[nano] éditeur de texte. il suffit d'ajouter le nom du fichier à ouvrir
	\end{description}
	

  \subsection{cd}
		
		\begin{description}
			\item[cd] retourne au répertoire personnel
			\item[cd ..] va au répertoire parent
			\item[cd items/sauvegarde] va dans le répertoire items/sauvegarde
			\item[cd /var/items/sauvegarde] va dans le répertoire items/sauvegarde
			\item[cd \textasciitilde pascal] va dans le répertoire personnel de pascal
		\end{description}
	

  \subsection{ls}
		
		\begin{description}
			\item[ls] liste les fichiers
			\item[ls -l] liste les fichiers avec les détails
			\item[ls -la] liste les fichiers avec les détails y compris les fichiers cachés
		\end{description}
	

  \subsection{cp}
		
		\begin{description}
			\item[cp a b] copie le fichier a en b
			\item[cp -f a b] écrase le fichier b avec le contenu du fichier a
			\item[cp -R d e] copie tout le répertoire d dans le répertoire e
		\end{description}
	

  \subsection{mv}
		
		\begin{description}
			\item[mv a b] déplace le fichier a en b
			\item[mv -f a b] déplace en écrasant le fichier b avec le contenu du fichier a
			\item[mv -R d e] déplace tout le répertoire en répertoire e
		\end{description}
	

\section{Utilitaires pour le texte}

  \subsection{nano}
		
		\begin{description}
			\item[nano fichier] ouvre le fichier fichier
		\end{description}

			\begin{itemize}
    \item Pour écrire dans un fichier ou le sauvegarder, utilisez Ctrl-o
    \item Pour quitter Nano, Ctrl-x
    \item Pour rechercher dans le fichier, Ctrl-w
\end{itemize}
	

		
		Pour le copier/coller, il faut au début du texte à copier faire CTRL-shift-6, puis flèche gauche/droite pour sélectionner le texte.
		
		La sélection finie il faut appuyer sur Alt-Shift-6.
		
		Pour le coller, il faut appuyer sur CTRL-u.
				

		
		Pour l'affichage de fichiers texte comme les logs il y a 4 commandes à se rappeler~:
		
		\begin{itemize}
			\item[head -<n>] affiche les n premières lignes d'un fichier
			\item[tail -<n>] affiche les n premières lignes d'un fichier
			\item[cat] affiche tout le fichier dans la console du haut vers le bas
			\item[tac] affiche tout le fichier dans la console du bas vers le haut
		\end{itemize}
				

  \subsection{more}
		
		Pour l'affichage de grands fichiers texte~: \emph{more} ou \emph{less} affiche le fichier 
		page par page (espace pour changer de page et q pour quitter)
